# FastFood Tasty - Frontend 🍔

Đây là mã nguồn phía giao diện (Client-side) của dự án **FastFood Tasty**, được xây dựng trên nền tảng React 19 và Vite.

## 💻 Công nghệ chính
- **React 19**: Sử dụng Functional Components và Hooks.
- **Vite**: Công cụ build siêu nhanh.
- **Material UI (MUI)**: Hệ thống giao diện đồng nhất và hiện đại.
- **Axios**: Giao tiếp với Backend API.
- **React Router Dom**: Quản lý điều hướng.
- **Chart.js & Recharts**: Hiển thị báo cáo doanh thu cho Admin.
- **React Slick**: Carousel hiển thị món ăn nổi bật.

## 🛠️ Cài đặt & Chạy
1. Cài đặt dependencies:
   ```bash
   npm install
   ```
2. Chạy môi trường phát triển:
   ```bash
   npm run dev
   ```
3. Truy cập: `http://localhost:5173`

## 📂 Cấu trúc thư mục
- `src/components`: Các component dùng chung.
- `src/pages`: Các trang chính (Home, Menu, Checkout, Admin, ...).
- `src/assets`: Hình ảnh và icon.
- `vercel.json`: Cấu hình deploy lên Vercel.
